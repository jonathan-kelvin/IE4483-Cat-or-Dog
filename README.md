## NTU IE4483: Data Mining and Artificial Intelligence Project
### Cat or Dog Image Classification

Cat or dog image classification with data augmentation and preprocessing, using custom CNN and ResNet50 transfer learning.

Final validation accuracy (averaged over 5 training rounds):
- CNN: 93.38% 
- ResNet50: 88.78%

Detailed project report in pdf file.
